🏗️ San Francisco Building Permits Analysis
📌 Project Overview
This project explores the Building Permits dataset from San Francisco. The goal is to analyze the time taken to issue permits, identify patterns across different permit types, and understand the financial implications (Estimated Costs).

📊 Key Insights
Speed of Issuance: The vast majority of permits are issued on the same day (OTC permits).

Significant Delays: New Construction projects take an average of 478 days to be issued.

Economic Value: New Construction, while slow, represents the highest financial investment in the city.

Data Handling: Managed skewed distributions and handled missing temporal data (NaT) to ensure accurate analysis.

🛠️ Tech Stack
Language: Python

Libraries: Pandas, Seaborn, Matplotlib
